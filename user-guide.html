🗺️ Overview

DACUM Lite is a browser-based tool for creating occupational analysis charts using the DACUM (Developing A CurriculUM) methodology. It supports the complete workflow from duty/task identification to document export — with no installation required.

📋
Table View
Classic list-based editor for entering duties and tasks with full drag-and-drop reordering.
🃏
Card View
Horizontal scrollable columns — one per duty — for a visual overview of all tasks.
🧱
Wall View
Full-screen display mimicking the traditional DACUM workshop wall with sticky-note cards.
↩️
Undo / Redo
Every action is reversible. Full command history with named snapshot versioning.
📤
Export
Export the complete DACUM chart as a Word document (.docx) or PDF file.
📁
Projects
Manage multiple DACUM charts in the sidebar. Each project is saved automatically.

🚀 Quick Start

Get your first DACUM chart ready in under 5 minutes:

1
Open DACUM LiteThe app loads directly in your browser. Your work is saved automatically.
2
Create a ProjectClick ＋ New Project in the sidebar. Give it a name — e.g. "Automotive Technician".
3
Fill Chart InfoEnter the Occupation Title, Job Title, and Scope of Work in the Chart Info tab.
4
Add Duties & TasksGo to Duties & Tasks. Click ➕ Add Duty — each duty starts with one task automatically.
5
Reorder with Drag & DropGrab the ⠿ handle on any duty or task to reorder. Labels update automatically.
6
Switch to Wall ViewClick 🧱 Wall to see your chart displayed as a workshop wall. Edit directly there.
7
ExportUse the toolbar to export as PDF or Word when ready.

📋 Chart Info Tab

The Chart Info tab stores all metadata about the occupation and the workshop. This information is included in exported documents.

📌 Scope of Work / Occupational Definition

Write a concise paragraph that describes what the occupation covers and where its boundaries lie. This is referenced to improve duty/task quality during AI-assisted generation.

💡 Good scopes include: key responsibilities, typical tools used, work environments, and what is explicitly excluded (e.g. managerial functions).

👥 Workshop Panel

The Workshop Panel section records the names of everyone present at the DACUM workshop:

  • Facilitators — The session leaders who guide the panel through the DACUM process
  • Observers — Individuals attending but not actively participating
  • Panel Members — The subject-matter experts who define duties and tasks
ℹ️ Enter one name per line in each field. All names are included in the exported Word document and PDF cover page.

✅ Duties & Tasks Tab

This is the core of DACUM Lite — where you build the occupational chart by defining duties and their associated tasks.

🔢 DACUM Standard Numbering

DACUM Lite automatically applies standard DACUM numbering to all duties and tasks:

Duty A
Execute Joinery Operations
Task A1
Cut mortise and tenon joints
Task A2
Route dadoes and grooves
Task A3
Drill holes for dowels

Duties are labelled A, B, C … Z, AA, AB… and tasks A1, A2, B1, B2… When you reorder anything via drag-and-drop, all labels update instantly.

⠿ Drag & Drop

Every duty and task has a ⠿ grip handle. Grab it to:

  • Reorder duties up or down (table view) / left or right (card/wall view)
  • Reorder tasks within the same duty
  • Move a task from one duty to another — just drop it over the target duty
⚠️ Drag using the ⠿ handle only — clicking anywhere else on the card starts editing the text.

🖥️ View Modes

Use the three-button switcher at the top of the Duties & Tasks tab to change the display:

📋 Table
🃏 Card
🧱 Wall

📋 Table View

The default view. Duties appear as vertical rows, tasks as indented items below each duty. Best for data entry and detailed editing.

🃏 Card View

Each duty becomes a vertical column with task cards arranged horizontally. Scroll left/right to see all duties. Good for reviewing the full chart width at a glance.

🧱 Wall View

Full-screen overlay that mimics the physical DACUM workshop wall. See the next section for full details.

💡 All edits made in any view are instantly reflected in the others. Your data is always in sync.

🧱 Wall View

Wall View is designed for live DACUM workshops. It displays all duties and tasks as sticky-note cards on a virtual wall — ideal for projecting in front of participants.

Card Action Icons

Each duty card (indigo) and task card (yellow) has small action icons in the top-right corner:

  • Duty card ＋ — Adds a new duty below
  • Duty card ✕ — Deletes the duty (with confirmation)
  • Task card ＋ — Adds a new task to that duty
  • Task card ✕ — Deletes the task immediately
ℹ️ All text in Wall View is editable — click any duty title or task text to edit it directly. Changes are saved to undo history.

📚 Additional Info Tab

Capture supplementary occupational information beyond the duty/task chart. All sections are included in exports.

  • Knowledge Requirements — What a worker must know
  • Skills Requirements — What a worker must be able to do
  • Worker Behaviors/Traits — Attitudes and personal attributes
  • Tools, Equipment, Supplies and Materials
  • Future Trends and Concerns
  • Acronyms — Terminology definitions
  • Career Path

You can rename any section heading by clicking ✏️ Rename, and add custom sections with ➕ Add Section.

📁 Projects

The sidebar on the left manages all your DACUM charts. Each project is stored independently in your browser's local storage.

＋
New ProjectClick ＋ New Project to create a blank chart.
✏️
RenameClick the pencil icon on any project to rename it.
🗑️
DeleteClick the bin icon to permanently remove a project.
💾
Export / ImportSave a project as a .json file and reload it later using Import Project.

📤 Export

Export the full DACUM chart from the main toolbar. Both formats include: cover page, Chart Info metadata, Workshop Panel, Scope of Work, all duties and tasks, and all Additional Info sections.

📄 Export as Word (.docx)

Generates a formatted Word document with a structured table layout. Compatible with Microsoft Word and LibreOffice.

🖨️ Export as PDF

Generates a printable PDF using jsPDF. The cover page includes all metadata and the workshop panel. Each duty appears as a clearly labelled section.

⚠️ Exports capture the current state of your chart. Make sure all duties, tasks, and metadata are complete before exporting.

📸 Snapshots

Before a major edit, save a named snapshot using the 📸 button in the toolbar. Snapshots let you restore any previous state of the chart with a single click.
